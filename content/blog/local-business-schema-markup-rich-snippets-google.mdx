### What is Schema Markup for Local Business?  

Schema markup is a code format (supported by Google, Bing, and Yahoo) that tells search engines exactly what your business does, when it’s open, and what customers are saying. Unlike generic HTML, schema uses a shared vocabulary like “LocalBusiness” or “Review” to make your data instantly scannable. For example, a yoga studio in London added “Service” schema for their “Postnatal Recovery” class, and now that service appears in Google’s “Top Stories” carousel for local searches.  

### Why is Schema Markup Important for Local Business?  

Local businesses need schema markup to dominate Google’s rich results, which get **2.5× more clicks** than standard listings (BrightEdge, 2024). Rich snippets include:  
- Star ratings and review counts (e.g., “4.8★ (123 reviews)”)

### How to Add Schema Markup to Your Local Business Website  

1. **Choose your schema type**  
   Use Google’s [Schema.org Generator](https://schema.org/) to pick the right format. A hair salon might use `https://schema.org/BeautySalon`, while a fitness studio would select `https://schema.org/Gym`.  

2. **Use a plugin or code manually**  
   - **WordPress**: Install Yoast SEO (£99/year) or Schema Pro (£149) to auto-generate markup.  
   - **Squarespace/Wix**: Use their built-in SEO tools (check for “Structured Data” settings).  
   - **Custom code**: Paste JSON-LD code into your `<body>` tag. Example for business hours:  
     ```json
     "openingHours": [
       "Mo,Tu,We,Th,Fr 08:00-18:00",
       "Sa 09:00-16:00"
     ]
     ```  

3. **Test and validate**  
   Run your URL through Google’s [Rich Results Test](https://search.google.com/test/rich-results) to catch errors. Fix any “Missing required fields” warnings within 48 hours to avoid indexing delays.

### Callout: Tip  

Pair schema markup with Google Business Profile updates. If your website says “Closed until 1pm,” but your GBP profile shows “Open 24/7,” Google will prioritize the GBP data — confusing customers and hurting trust.  

### Callout: Warning  

Avoid duplicate schema on the same page. For example, don’t add both `Review` and `AggregateRating` without linking them. Google penalizes “over-optimized” pages with a 30%+ drop in rankings.
