## 🟡 Backlog

### [TASK-001] Create Claude Code Skills for GitHub Traffic
**Priority**: Medium | **Category**: SEO / Growth / Developer Marketing

**Goal**: Build a collection of free, useful Claude Code skill files (`.md` format) hosted in a dedicated GitHub repo, similar to `claude-seo.md/skills`. Each skill targets developers and marketers who use Claude Code — they discover the skills, use them, and see DataLatte.pro as the source/creator.

**Why it works**:
- GitHub repos get indexed by Google — each skill file = a targetable keyword page
- Developers share useful Claude skills → organic backlinks & social mentions
- Skills can be niche-marketing-focused (local SEO audits, Google Ads copy, etc.) → direct audience match
- README links to datalatte.pro as the creator → referral traffic + brand authority

**Proposed skills to create**:

| Skill file | Description | Target keyword |
|---|---|---|
| `local-seo-audit.md` | Audits a local business website for Local SEO gaps | "local seo audit claude skill" |
| `google-business-profile-optimizer.md` | Generates optimized GBP descriptions, posts, Q&As | "google business profile claude" |
| `google-ads-copy.md` | Writes Google Ads headlines/descriptions for local businesses | "google ads copy generator claude" |
| `meta-ads-copy.md` | Writes Facebook/Instagram ad copy for local niches | "meta ads claude skill" |
| `review-response-writer.md` | Drafts professional responses to Google/Yelp reviews | "review response generator claude" |
| `local-keyword-research.md` | Generates local keyword clusters for a niche + city | "local keyword research claude" |
| `email-campaign-writer.md` | Writes email sequences for local service businesses | "email marketing claude skill" |
| `competitor-analysis.md` | Analyzes local competitor websites and GBP profiles | "competitor analysis local business" |

**Implementation steps**:
1. Create new GitHub repo: `natalyineu/claude-marketing-skills`
2. Write each skill as a `.md` file following Claude Code skill conventions
3. Create a polished README with overview, usage instructions, and link to datalatte.pro
4. Add a `skills.json` index file for discoverability
5. Submit repo to Claude Code community resources / directories
6. Write a blog post on datalatte.pro about using Claude for local marketing → links to the repo
7. Cross-post to Reddit (r/ClaudeAI, r/LocalSEO), Indie Hackers, Twitter/X

**Success metrics**:
- GitHub repo stars / forks
- Referral traffic from github.com in Google Analytics
- Backlinks from repo README appearing in Search Console
